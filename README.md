# Генератор осмысленных предложений на основе формальной грамматики

> Можно генерировать предложения по заданной грамматике на английском языке, а также получать их перевод на русский


## Как собрать `.exe` 

Чтобы получить готовое исполняемое приложение (`gui_generator_pyqt.exe`), выполните следующие шаги:
### 1. Установите PyInstaller

```bash
pip install pyinstaller
```
### 2. Соберите приложение

```bash
pyinstaller gui_generator_pyqt.py --onefile --windowed --collect-all pronouncing --collect-all cmudict --copy-metadata cmudict
```


## После запуска приложения:

### 1. Загрузите грамматику: 
- В меню **≡** выберите "Загрузить грамматику"  
- Загрузите файл **`grammar.txt`**

### 2. Загрузите словарь слов: 
- Аналогично выберите "Загрузить терминалы"  
- Далее загрузите файл **`terminals.txt`**

### 3. Нажмите на кнопку "Сгенерировать предложение"


### Дополнительно:
- **Очистить** — очистит оба поля (оригинал + перевод)
- **Копировать предложение** — скопирует английское предложение в буфер обмена
- **Копировать перевод** — скопирует русский перевод
